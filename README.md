# Online Proctoring Tool

This is a repo for in-browser online proctoring using Tensorflow.js and React


### Models being used:
- pre-trained [COCO SSD](https://github.com/tensorflow/tfjs-models/tree/master/coco-ssd) 
- [WIP] BlazeFace

### Requirements
simple web server
